Hyper Bean

Repo

A coffee shop ordering web app with separate admin and user roles. Users can create an account, order coffee, add funds, and view their transaction history. Admins can manage users, control menu items and add-ons, and update item availability. The system uses a C# backend with SQLite and a simple frontend built with Vanilla JavaScript. This project serves as my first complete CRUD-based web application on a localhost.
